## Featuring
- JetPack
  - Compose
  - Navigation Component
  - Hilt
  - Preferences Datastore
  - Room
  - Work Manager
- Arrow
- Firebase
  - Authentication
  - Firestore

## Requirements
Android Snow Cone (API level 31) or later

## Project setup
You need to configure a [Firebase project](https://firebase.google.com/docs/android/setup)
